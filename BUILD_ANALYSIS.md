## 🔧 Troubleshooting

### Build Fails
1. Check Node.js version: `node --version` (v14+)
2. Clear cache: `npm cache clean --force`
3. Reinstall: `rm -rf node_modules && npm install`

### Dependency Issues
1. Check `package.json` for custom scripts
2. Review `npm run` to see available commands
3. Check for peer dependency warnings

### Deploy Fails
1. Verify `dist/` directory exists
2. Check for production build errors
3. Validate all assets are included

## 📈 Performance Optimization

After build, optimize with:
- **Minification**: Enabled in production build
- **Code Splitting**: Automatic with webpack/vite
- **Asset Compression**: Use gzip for served files
- **CDN**: Use CDN for static files
